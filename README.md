use the `sudo` comment
```
	apt-get update
	apt-get upgrade
	apt-get dis-upgrade
```

安裝編譯器
	`apt-get install default-jdk`

檢查版本
	`java -version`

環境安裝完畢可以開始編譯 .java 
	`javac abc.java`

執行編譯過的檔案 .class 
	`java abc`

載入外部數據 .txt
	`java abc < efg.txt`

思考流程 => 窮舉；逼近；消去

編輯說明檔案
	`gedit README.md`

push to github
	`git add .`
	`git commit -m '說明文字'`
	`git push`
